Методика и ограничения

  • Основной период — полные календарные годы 2016–2025; строки 2014–2015 и неполный 2026 год исключены.
  • ПГПЖ-75 рассчитана как сумма max(75 − возраст, 0). Площадь используется только для аддитивных показателей.
  • Для всего периода цвет изменения сравнивает среднегодовое значение 2016–2020 и 2021–2025; для отдельного года — выбранный год с предыдущим.
  • Средний возраст рассчитан по строкам с валидным возрастом 0–110 лет. Он описывает умерших, а не возрастной риск населения.
  • Рубрики с n<5 объединяются на третьем уровне в группу малых чисел и не подходят для устойчивых выводов.
